# Simple Quotes Scraper

A lightweight, efficient Python script that scrapes quotes, authors, and associated tags from [Quotes to Scrape](https://quotes.toscrape.com/). This project serves as an excellent introduction to web scraping basics using `requests` and `BeautifulSoup`.

## Features

- **Custom User-Agent Headers:** Mimics a standard browser request to avoid basic bot blocks.
- **HTML Parsing:** Leverages `BeautifulSoup` to easily navigate and extract structured data from DOM elements.
- **Clean Console Output:** Formats the scraped text cleanly in the terminal for easy reading.
